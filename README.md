# TuaObra para Pedreiro

**TuaObra para Pedreiro** é um aplicativo Android desenvolvido para conectar pedreiros e clientes de forma prática e eficiente. Com ele, os pedreiros podem se cadastrar, visualizar as demandas dos clientes e salvar os contatos como principais funcionalidades. O app atua como um **front-end** para a API RESTful **TuaObra**, facilitando a comunicação e o gerenciamento das demandas de obras.

---

## Funcionalidades Principais

- **Cadastro de Pedreiros**: Permite aos pedreiros se cadastrarem no sistema.
- **Consultas de Demandas**: Visualize as demandas de clientes para serviços de construção e reparos.
- **Armazenamento de Contatos**: Salve os contatos dos clientes e tenha fácil acesso a informações importantes.
- **Integração com a API**: O app se comunica com a API RESTful **TuaObra**, garantindo  integração eficiente back-end e o front-end.
